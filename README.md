For termux and kali linux
```
git clone https://github.com/yuhertz/ddos-v5
cd ddos-v5
pip install -r requirements.txt
python3 ddos.py
```

For install python, git, pip

(for kali linux)
```
sudo apt install git
sudo apt install python3
sudo apt install pip
```

(for termux)
```
install git
install python
install pip
```
